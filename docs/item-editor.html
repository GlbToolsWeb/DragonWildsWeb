ITEM EDITOR
How do I use this?

Click on the plus (+) icon in the Player Inventory Section and select your Save file.

Make changes, Add Stuff, Remove Stuff, manually set the quantity of an item to 1000000... then hit Save!


Your Save file is here:

C:\Users\YourComputerName\AppData\Local\RSDragonwilds\Saved\SaveCharacters\


Disclaimer:

Data is compiled from the latest game files automatically.

I don't check if the file is a developers only item or a test asset.

I compile it all and give you access to everything.

This means you should always back up your save.

It is possible to inject something that isn't allowed and it crashes.


According to the game files, this is every item in the game, including dev/test items.
